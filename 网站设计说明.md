
### 3.2 首页各区块说明

#### Hero 区（头图）
- **作用**：第一眼吸引访客，介绍自己
- **内容**：名字 + 职位 + 一句话介绍 + 两个按钮
- **按钮**：「查看作品」跳到作品区，「联系我」跳到联系区

#### 作品展示区（WORKS）
- **显示数量**：4个（主页精华）
- **卡片内容**：封面图 + 日期 + 标题（**不显示描述**，保持简洁）
- **交互**：鼠标悬停卡片放大，点击进入详情

#### 文章列表区（ARTICLES）
- **显示数量**：3篇（主页精华）
- **样式**：时间轴（左侧日期，右侧标题）
- **卡片内容**：封面图 + 日期 + 标题（**不显示描述**）

#### 常用工具区（TOOLS）
- **显示数量**：最多10个
- **样式**：网格布局，图标 + 工具名
- **交互**：点击跳转到工具官网

#### 联系我区（CONTACT）
- **内容**：邮箱 + 社交平台链接（GitHub、微信等）

### 5.2 管理后台功能清单

#### 数据概览
- 显示：作品数量、文章数量、工具数量、访问量
- 作用：一眼看到网站内容全貌

#### 作品管理
- **新增**：填标题、描述、上传封面（或贴图片链接）、填链接
- **编辑**：点击「编辑」按钮，修改信息
- **删除**：点击「删除」按钮，确认后删除
- **图片上传**：支持两种方式
  1. 点击上传区域，选择文件
  2. 粘贴图片链接（推荐用 imgbb.com 图床）

#### 文章管理
- 同上，多了「发布日期」字段

#### 工具管理
- 同上，多了「分类标签」字段

#### 数字分身
- **基础配置**：名字、头像、欢迎语
- **预设话术**：设置自动回复内容
- **快捷回复**：设置快速回复按钮

#### 基础配置
- 修改网站标题、简介、联系方式等

### 5.3 图片上传的两种方式

```
方式1：文件上传             方式2：粘贴链接
┌──────────────┐        ┌─────────────────────────────┐
│  点击上传区域  │        │  或粘贴图片链接              │
│  选择文件      │        │  ┌───────────────────────┐  │
│  自动上传      │        │  │ https://i.ibb.co/...   │  │
│  显示预览      │        │  └───────────────────────┘  │
└──────────────┘        └─────────────────────────────┘
         ↓                         ↓
    适合本地开发              适合线上部署（Vercel）
```

> **为什么两种方式？** 因为 Vercel 不支持文件上传（只读文件系统），所以线上部署后要用「粘贴链接」方式。

## 六、配色方案

### 6.1 主色调

| 颜色 | 色值 | 用途 |
|------|------|------|
| 🔵 主蓝色 | `#4facfe` | 按钮、链接、强调色 |
| 🔵 次蓝色 | `#00f2fe` | 渐变、副强调色 |
| ⚪ 背景白 | `#ffffff` | 页面背景 |
| ⚪ 半透明白 | `rgba(255,255,255,0.82)` | 导航栏背景（毛玻璃效果）|
| ⚫ 主文字 | `#1a1a2e` | 标题、正文 |
| ⚫ 次文字 | `rgba(26,26,46,0.45)` | 副标题、说明文字 |

### 6.2 渐变效果

```css
/* Hero 区背景渐变 */
background: linear-gradient(135deg, #4facfe 0%, #00f2fe 100%);

/* 卡片悬停阴影 */
box-shadow: 0 8px 32px rgba(79,172,254,0.15);
```

### 6.3 为什么要这样配色？

- **蓝色**：代表专业、科技、信任，适合设计师/开发者
- **白色**：简洁干净，突出重点内容
- **毛玻璃效果**：现代感，让导航栏不遮挡背景

## 八、交互设计

### 8.1 导航栏

- **固定顶部**：滚动时一直显示在顶部
- **毛玻璃效果**：半透明 + 模糊，不遮挡背景
- **高亮当前区域**：滚动到哪个区块，导航栏对应菜单高亮
- **管理按钮**：右上角蓝色边框按钮，点击打开管理后台

### 8.2 作品卡片交互

```
鼠标悬停卡片
    ↓
1. 卡片向上移动 4px（translateY(-4px)）
2. 阴影变深（box-shadow 加深）
3. 图片稍微放大（scale(1.05)）
    ↓
点击卡片
    ↓
跳转到作品详情或外部链接
```

### 8.3 聊天面板（数字分身）

```
1. 点击右下角聊天图标
    ↓
2. 弹出聊天面板
    ↓
3. 显示欢迎语 + 快捷回复按钮
    ↓
4. 用户输入消息 → 自动回复预设话术
    ↓
5. 点击右上角 ✕ 关闭面板
```

### 8.4 管理后台交互

```
1. 打开 admin.html
    ↓
2. 输入账号密码登录
    ↓
3. 左侧菜单切换不同管理页面
    ↓
4. 点击「新增」→ 弹出表单弹窗
    ↓
5. 填写信息 → 点击「保存」
    ↓
6. 显示「保存成功」提示（Toast 弹窗）
```

## 十、SEO 和优化

### 10.1 已优化的地方

- ✅ 语义化 HTML（`<header>` `<nav>` `<main>` `<footer>`）

## 十二、日常维护指南

### 12.1 新增一篇作品

1. 打开 `https://yan-portfolio-omega.vercel.app/admin.html`
2. 登录
3. 点击左侧「作品管理」
4. 点击「+ 新增作品」
5. 填写标题、描述
6. 上传封面图片 **或** 粘贴图片链接（从 imgbb.com 获取）
7. 填写作品链接（可选）
8. 点击「保存」

### 12.2 修改网站标题/简介

1. 管理后台 → 左侧「基础配置」
2. 修改网站名称、简介、联系方式
3. 点击「保存基础配置」

### 12.3 图片链接获取步骤（重要！）

因为 Vercel 不支持直接上传图片，所以要用图床：

```
第1步：打开 https://imgbb.com
第2步：点击「Start uploading」上传图片
第3步：上传完成后，右键点击图片 →「复制图像地址」
第4步：粘贴到管理后台的「粘贴图片链接」输入框
```

> ⚠️ 注意：一定要复制 `https://i.ibb.co/...` 开头的直链，不是 `https://ibb.co/...` 的页面链接！

## 十三、常见问题 FAQ

### Q1：为什么管理后台登录失败？
**A**：可能是 Redis 里没有管理员账号数据。运行 `seed.js` 重新导入数据。

### Q2：为什么图片显示不出来？
**A**：检查图片链接是否是 `https://i.ibb.co/...` 开头的直链，不是 `http://` 或页面链接。

### Q3：为什么主页和 all.html 显示的数据不一样？
**A**：之前的问题，现在代码已修复。确保 `IS_SERVER` 判断正确，线上会调 API 获取最新数据。

### Q4：如何在本地预览网站？
**A**：打开 PowerShell，运行：
```powershell
cd C:\Users\47910\Documents\Yan_Project\Workbuddy
node server.js
```
然后浏览器打开 `http://localhost:3000`

### Q5：如何更新线上网站？
**A**：修改代码后，推送到 GitHub：
```powershell
git add -A
git commit -m "更新说明"
git push origin main
```
Vercel 会自动重新部署。
